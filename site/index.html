About This Project

This project showcases an authenticated FastAPI login system implementing role-based access control (RBAC). It demonstrates user signup with secure password hashing, JWT access and refresh tokens stored in HTTP-only cookies, and protected admin/user routes with redirects based on role. Use the Signup link to create an account, then Login to be redirected to the appropriate dashboard.
